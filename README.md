- Pokémon Simulador de Batalha

Um simulador de batalha por turnos desenvolvido em Linguagem C.
Este projeto foi criado para aplicar conceitos fundamentais da Ciência da Computação, como alocação de memória, estruturação de dados e lógica booleana, servindo de base para estudos futuros.

---

- Sobre o Projeto

O programa permite que o usuário cadastre dois Pokémon com atributos personalizados (Vida, Ataque, Defesa) e simule uma batalha automática entre eles. O sistema conta com validações robustas para impedir dados inconsistentes (como vida negativa ou nula).

- Objetivos de Aprendizado
* Structs: Modelagem de entidades do mundo real (Objeto Pokémon) em C.
* Ponteiros: Manipulação direta de memória e passagem por referência nas funções.
* Input Validation: Uso de laços `do-while` para garantir a integridade dos dados inseridos pelo usuário.
* Lógica Booleana: Controle de fluxo complexo nos menus de navegação.

---

- Funcionalidades

 * Cadastro de Pokémon: Input de nome e atributos via terminal.
 * Sistema de Validação: Impede que o usuário digite valores negativos ou zero para os atributos.
 * Batalha Automática: Algoritmo que calcula o dano baseado na diferença entre Ataque e Defesa.
 * Menu Interativo: Navegação entre as opções de "Batalhar" e "Apresentar Dados".
